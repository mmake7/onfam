1

사이트 접속 방법

비온팜 홈페이지와 관리자 화면에 접속하는 방법을 알려드립니다.
인터넷 브라우저(크롬, 엣지 등)만 있으면 누구나 접속할 수 있습니다.

1-1

인터넷 브라우저 열기

컴퓨터 바탕화면이나 작업 표시줄에서 인터넷 브라우저 아이콘을 찾아 두 번 클릭합니다.

info 추천 브라우저
크롬 (Chrome) 엣지 (Edge)
1-2

홈페이지 주소 입력하기

브라우저 맨 위 주소 입력란(긴 흰색 막대)을 클릭한 뒤, 아래 주소를 입력하고 Enter 키를 누릅니다.

https://www.beeonfarm.co.kr

※ 주소를 정확히 입력해야 합니다. 오타가 나면 페이지가 열리지 않습니다.

1-3

관리자 화면으로 이동하기

홈페이지가 열리면, 주소 뒤에 /admin을 추가로 입력하여 관리자 전용 화면으로 이동합니다.

https://www.beeonfarm.co.kr/admin
warning

주의: 관리자 화면은 권한이 있는 사람만 접속할 수 있습니다. 로그인이 필요합니다.

photo_camera [스크린샷 자리] 브라우저 주소창 예시 화면
2

회원가입 및 로그인

관리자 화면을 사용하려면 먼저 계정을 만들고 로그인해야 합니다.
이미 계정이 있으면 2-2. 로그인하기로 바로 넘어가세요.

2-1

회원가입하기 (처음 사용자)

①

관리자 화면 접속 후, 로그인 페이지에서 회원가입 버튼을 클릭합니다.

②

이름, 이메일 주소, 비밀번호를 입력란에 차례대로 적습니다.

③

비밀번호는 8자 이상으로, 영어와 숫자를 섞어서 만들어 주세요.

④

모든 칸을 채운 뒤 가입하기 버튼을 클릭합니다.

⑤

"가입이 완료되었습니다"라는 메시지가 나오면 성공입니다.

2-2

로그인하기

①

로그인 화면에서 가입할 때 사용한 이메일 주소와 비밀번호를 입력합니다.

②

로그인 버튼을 클릭합니다.

③

관리자 대시보드 화면이 나타나면 로그인 성공입니다.

2-3

비밀번호를 잊어버렸을 때

①

로그인 화면에서 비밀번호 찾기 글자를 클릭합니다.

②

가입할 때 사용한 이메일 주소를 입력하고 전송을 누릅니다.

③

이메일을 확인하여 안내에 따라 새 비밀번호를 설정합니다.

photo_camera [스크린샷 자리] 로그인 화면 예시
3

관리자 대시보드 사용법

로그인하면 가장 먼저 보이는 화면이 대시보드입니다.
대시보드에서는 홈페이지의 전체 현황을 한눈에 볼 수 있고, 왼쪽 메뉴를 통해 회원 관리, 신청 현황, 콘텐츠 관리 등 모든 기능에 접근할 수 있습니다.

3-2

왼쪽 사이드바 메뉴 — 각 항목 역할

화면 왼쪽의 어두운 색 세로 막대가 사이드바 메뉴입니다. 원하는 항목을 클릭하면 해당 관리 화면으로 바로 이동합니다.
현재 보고 있는 메뉴는 노란색 하이라이트로 표시됩니다.

dashboard
대시보드 첫 화면

로그인 직후 자동으로 표시되는 전체 현황 요약 화면입니다. 통계 카드와 최근 활동을 한눈에 확인합니다.

people
회원 관리

가입된 모든 회원의 목록을 볼 수 있습니다. 이름·이메일로 검색하거나, 회원의 역할(관리자/일반)을 변경할 수 있습니다.

assignment
신청 현황

교육 프로그램에 신청한 사람들의 신청 내역과 상태(대기중/결제완료/취소/환불)를 확인합니다. 프로그램별로 필터링할 수 있습니다.

article
콘텐츠 관리

홈페이지에 표시되는 게시글(콘텐츠)을 작성·수정·삭제합니다. 한국어/영어 이중 언어 관리와 게시 순서 변경이 가능합니다.

photo_library
이미지 관리

사진 파일을 업로드하거나 이미 올린 사진을 삭제합니다. JPG/PNG/WebP 형식, 5MB 이하 파일만 가능합니다.

payments
결제 관리

모든 결제 내역을 조회합니다. 날짜·상태별로 필터링하고, 필요시 결제 취소(환불) 처리를 할 수 있습니다.

settings
설정

사이트 기본 정보(제목, 설명 등)를 변경합니다.

4

게시글 작성 / 수정 / 삭제

홈페이지에 보여지는 글(게시글)을 새로 쓰거나, 이미 올린 글의 내용을 고치거나, 필요 없는 글을 지우는 방법을 알려드립니다.

add_circle 새 글 쓰기 edit 글 수정하기 delete 글 삭제하기
4-1

새 글 쓰기

홈페이지에 새로운 글을 올리는 방법

①

왼쪽 메뉴에서 게시글 관리를 클릭합니다.

②

화면 오른쪽 위의 + 새 글 쓰기 버튼을 클릭합니다.

③

글 작성 화면이 나타납니다. 아래 항목을 채워주세요:

제목 — 글의 제목을 입력합니다
카테고리 — 글의 종류를 선택합니다 (예: 공지, 프로그램 등)
내용 — 본문 내용을 작성합니다
대표 이미지 — 글과 함께 보일 사진을 선택합니다 (선택사항)
④

모든 내용을 입력한 뒤, 아래쪽의 저장하기 버튼을 클릭합니다.

⑤

"저장되었습니다"라는 메시지가 나오면 글이 성공적으로 올라간 것입니다.

4-2

글 수정하기

이미 올린 글의 내용을 고치는 방법

①

②

글 상세 화면에서 수정하기 버튼을 클릭합니다.

③

바꾸고 싶은 부분을 수정한 뒤 저장하기를 클릭합니다.

이미지 업로드 방법

홈페이지에 사진(이미지)을 올리는 방법을 알려드립니다.
게시글 작성 중에 사진을 넣거나, 이미지 관리에서 따로 올릴 수 있습니다.

5-1

게시글에 사진 넣기

①

글 작성 화면에서 이미지 추가 버튼(사진 모양 아이콘)을 클릭합니다.

②

파일 선택 창이 나타나면, 내 컴퓨터에서 올리고 싶은 사진 파일을 찾아 선택합니다.

③

열기를 클릭하면 사진이 업로드됩니다.

④

업로드가 완료되면 글 안에 사진이 표시됩니다.

5-2

aspect_ratio

JPG, PNG, WebP

sd_storage

최대 크기

파일당 5MB 이하

photo_size_select_large

권장 크기

가로 1200px 이상

5-3

올린 사진 삭제하기

①

②

③

확인 창에서 삭제를 클릭하면 사진이 지워집니다.

tips_and_updates

꿀팁: 사진 크기 줄이는 방법

사진 파일이 너무 크면 업로드가 안 될 수 있습니다. 이럴 때는 사진을 오른쪽 클릭 → 편집하여 크기를 줄이거나, 무료 사이트(iloveimg.com 등)에서 사진 크기를 줄인 뒤 다시 올려보세요.

6

자주 묻는 질문 (FAQ)

관리자들이 자주 질문하는 내용을 모아두었습니다.
아래에서 비슷한 상황을 찾아 해결 방법을 확인해 보세요.

Q

비밀번호가 기억나지 않아요.

A

로그인 화면에서 "비밀번호 찾기"를 클릭하고, 가입할 때 사용한 이메일 주소를 입력해 주세요. 이메일로 비밀번호 재설정 안내가 발송됩니다. 자세한 방법은 2-3. 비밀번호를 잊어버렸을 때를 참고하세요.

Q

글을 저장했는데 홈페이지에 안 보여요.

A

먼저 홈페이지를 새로고침(키보드 F5 키 또는 Ctrl+R)해 보세요. 그래도 안 보인다면, 글의 상태가 "공개"로 되어 있는지 확인해 주세요. "비공개"로 설정된 글은 홈페이지에 표시되지 않습니다.

Q

사진이 업로드가 안 돼요.

A

다음 사항을 확인해 주세요:
① 파일 크기가 5MB 이하인지 확인합니다.
② 파일 형식이 JPG, PNG, WebP 중 하나인지 확인합니다.
③ 인터넷 연결이 정상인지 확인합니다.
위 방법으로도 해결되지 않으면 관리자에게 문의해 주세요.

Q

삭제한 글을 되살릴 수 있나요?

A

안타깝지만, 한 번 삭제한 글은 되살릴 수 없습니다. 삭제하기 전에 정말 지워도 괜찮은지 꼭 확인해 주세요. 중요한 글은 삭제 대신 "비공개"로 상태를 바꿔 숨기는 것을 추천합니다.

Q

관리자 화면이 갑자기 안 열려요.

A

① 인터넷 연결이 되어 있는지 확인합니다.
② 브라우저의 주소가 정확한지 다시 한번 확인합니다.
③ 브라우저를 완전히 껐다가 다시 열어봅니다.
④ 그래도 안 된다면 다른 브라우저(크롬, 엣지)로 시도해 보세요.

Q

글을 수정했는데 이전 내용으로 보여요.

A

브라우저가 이전 페이지를 기억하고 있을 수 있습니다. Ctrl + Shift + R 키를 동시에 눌러 "강력 새로고침"을 하면 최신 내용이 표시됩니다.
