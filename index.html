A

새 글 작성

홈페이지에 새로운 콘텐츠(게시글)를 올리는 전체 과정

홈페이지에 보여지는 새로운 글을 작성하는 방법을 단계별로 안내합니다.
콘텐츠 관리 메뉴 진입부터 게시 완료까지, 모든 과정을 따라 해 보세요.

A-1

콘텐츠 관리 메뉴로 이동하기

①

관리자 화면에 로그인하면 대시보드가 나타납니다.

②

화면 왼쪽 메뉴에서 게시글 관리 (또는 콘텐츠 관리) 항목을 찾아 클릭합니다.

info 왼쪽 메뉴 위치 안내
article
게시글 관리 — 이 메뉴를 클릭하세요
③

기존에 작성된 게시글 목록이 표(테이블) 형태로 나타납니다.

photo_camera [스크린샷] 왼쪽 메뉴에서 '게시글 관리' 클릭 위치 빨간 동그라미로 클릭할 메뉴 위치 표시
A-2

'새 글 작성' 버튼 찾아 클릭하기

①

게시글 목록 화면의 오른쪽 위에 있는 버튼을 찾아 클릭합니다:

add 새 글 작성

※ 위와 같은 초록색 버튼을 찾으세요.

②

버튼을 클릭하면 글 작성 화면(빈 양식)이 열립니다.

photo_camera [스크린샷] 게시글 목록 화면 — '새 글 작성' 버튼 위치 오른쪽 위 초록색 버튼에 빨간 동그라미 표시
A-3

각 입력란 채우기

글 작성 화면에는 여러 입력란이 있습니다. 위에서 아래 순서대로 채워주세요.

① 제목 (필수)

글의 제목을 입력하는 칸입니다. 홈페이지 방문자가 가장 먼저 보는 부분이므로 한눈에 내용을 알 수 있는 제목으로 짧고 명확하게 작성하세요.

입력 예시: 2026년 봄맞이 꿀벌 체험 프로그램 안내
② 카테고리 (필수)

글의 종류(분류)를 선택하는 드롭다운 메뉴입니다. 칸을 클릭하면 목록이 펼쳐지며, 해당하는 분류를 하나 선택합니다.

공지사항 프로그램 소식 자유게시판 등
③ 본문 내용 (필수)

글의 본문(내용)을 작성하는 넓은 입력 영역입니다. 글꼴 굵게, 기울임, 목록 등의 서식 도구(툴바)를 이용해 보기 좋게 꾸밀 수 있습니다.

format_bold format_italic format_underlined | format_list_bulleted format_list_numbered | image link

본문 내용을 여기에 작성합니다...

※ 위 아이콘 모양의 도구 바에서 글꼴 서식을 변경할 수 있습니다.

④ 대표 이미지 (선택사항) (선택)

글과 함께 보여질 대표 사진(썸네일)을 선택합니다. 이 사진은 글 목록에서 미리보기로 표시됩니다. 꼭 넣지 않아도 괜찮습니다.

이미지 첨부 방법은 아래 A-4 단계에서 자세히 안내합니다.

photo_camera [스크린샷] 글 작성 화면 — 제목, 카테고리, 본문, 이미지 입력란 각 입력란 옆에 번호(①~④)로 위치 표시
A-4

이미지(사진) 첨부하기

글에 사진을 넣고 싶을 때, 아래 순서대로 따라 하세요.

①

글 작성 화면에서 이미지 추가 버튼 또는 파일 선택 버튼을 클릭합니다.

cloud_upload

이미지를 끌어다 놓거나

또는 파일 선택을 클릭하세요

※ 위와 비슷한 영역이 화면에 보입니다.

②

파일 선택 창이 열리면, 내 컴퓨터에서 올리고 싶은 사진 파일을 찾아 클릭하고 열기 버튼을 누릅니다.

③

사진이 업로드되면서 미리보기가 표시됩니다. 올라간 사진이 맞는지 확인하세요.

warning 사진 올리기 전 확인사항
aspect_ratio

파일 형식

JPG, PNG, WebP

sd_storage

최대 크기

파일당 5MB 이하

photo_size_select_large

권장 크기

가로 1200px 이상

photo_camera [스크린샷] 이미지 첨부 영역 — 파일 선택 버튼 클릭 위치 파일 선택 버튼에 빨간 동그라미 표시
A-5

'저장' 또는 '게시' 버튼 클릭하기

①

모든 내용을 입력했으면, 화면 아래쪽의 버튼을 확인합니다.

②

목적에 맞는 버튼을 클릭하세요:

publish 게시하기

글이 바로 홈페이지에 공개됩니다. 방문자가 즉시 볼 수 있습니다.

save 임시 저장

글이 저장되지만 홈페이지에는 보이지 않습니다. 나중에 게시할 수 있습니다.

③

"저장되었습니다" 또는 "게시되었습니다"라는 메시지가 나오면 성공입니다!

A-6

게시된 글 확인하기

①

방법 1 — 관리자 화면에서 확인:

게시글 관리 목록으로 돌아가면, 방금 작성한 글이 목록 가장 위에 나타납니다.

②

방법 2 — 실제 홈페이지에서 확인:

새 브라우저 탭을 열고 홈페이지 주소로 접속하여 글이 정상적으로 보이는지 확인합니다.

tips_and_updates

꿀팁: 홈페이지에서 글이 안 보이면 키보드의 F5 키를 눌러 새로고침해 보세요. 그래도 안 보이면 글의 상태가 "공개"인지 확인해 주세요.

photo_camera [스크린샷] 게시글 목록에서 새로 올린 글 확인 목록 상단에 새 글이 표시된 화면
check_circle

새 글 작성 완료!

축하합니다! 새 콘텐츠(게시글)가 성공적으로 등록되었습니다.

B

글 수정

이미 올린 콘텐츠(게시글)의 내용을 수정하는 방법

이미 게시한 글의 제목, 본문, 카테고리, 이미지 등을 수정하는 방법을 안내합니다.
오타 수정부터 내용 보강까지, 간단한 절차로 수정할 수 있습니다.

B-1

수정하고 싶은 글 찾기

①

왼쪽 메뉴에서 게시글 관리를 클릭하여 글 목록으로 이동합니다.

②

글 목록에서 수정하고 싶은 글을 찾습니다. 아래 방법을 이용하면 더 빠르게 찾을 수 있습니다:

search 검색 기능

목록 위쪽의 검색창에 글 제목의 일부를 입력하면 빠르게 찾을 수 있습니다.

filter_list 카테고리 필터

카테고리별로 필터링하면 해당 분류의 글만 볼 수 있습니다.

③

수정할 글의 제목을 클릭합니다. 글 상세 화면이 열립니다.

photo_camera [스크린샷] 게시글 목록 — 검색창과 글 제목 클릭 위치 검색창과 수정할 글 제목에 빨간 동그라미 표시
B-2

'수정하기' 버튼 클릭하기

①

글 상세 화면 또는 글 목록에서 수정하기 버튼을 찾아 클릭합니다.

info 수정 버튼 위치 (두 가지)
글 상세 화면 → 화면 오른쪽 위 또는 아래에 위치
글 목록 → 해당 글 오른쪽의 edit 아이콘 클릭
②

글 작성 때와 동일한 편집 화면이 열리며, 기존에 입력했던 내용이 모두 채워져 있습니다.

photo_camera [스크린샷] 글 상세 화면 — '수정하기' 버튼 위치 수정하기 버튼에 빨간 동그라미 표시
B-3

내용 수정하기

①

바꾸고 싶은 부분을 클릭하여 내용을 수정합니다. 제목, 카테고리, 본문, 이미지 모두 수정 가능합니다.

②

수정할 수 있는 항목들:

title 제목 변경
category 카테고리 변경
edit_note 본문 내용 변경
image 이미지 변경/삭제
③

기존 이미지를 바꾸려면, 기존 이미지 옆의 X 버튼으로 삭제한 뒤, 새 이미지를 업로드합니다.

B-4

수정 완료 후 저장하기

①

수정을 모두 마친 뒤, 화면 아래의 저장하기 버튼을 클릭합니다.

②

"수정되었습니다" 메시지가 나오면 수정이 완료된 것입니다.

③

홈페이지에서 수정된 내용이 바로 반영되었는지 확인합니다. 안 바뀌어 보이면 Ctrl + Shift + R (강력 새로고침)을 눌러보세요.

photo_camera [스크린샷] 글 수정 화면 — 수정 후 '저장하기' 버튼 클릭 위치 저장하기 버튼에 빨간 동그라미 표시
check_circle

글 수정 완료!

수정한 내용이 성공적으로 반영되었습니다.

C

글 삭제

더 이상 필요 없는 콘텐츠(게시글)를 삭제하는 방법

warning

삭제 전 꼭 읽어주세요!

  • • 삭제한 글은 되돌릴 수 없습니다. 한 번 삭제하면 복구가 불가능합니다.
  • • 삭제하기 전에 정말 지워도 되는 글인지 한 번 더 확인해 주세요.
  • • 임시로 숨기고 싶다면, 삭제 대신 글의 상태를 "비공개"로 변경하는 것을 추천합니다.
C-1

삭제할 글 찾기

①

왼쪽 메뉴에서 게시글 관리를 클릭하여 글 목록으로 이동합니다.

②

삭제하려는 글을 목록에서 찾습니다. (검색이나 카테고리 필터를 활용할 수 있습니다.)

C-2

'삭제' 버튼 클릭하기

①

삭제할 글의 오른쪽에 있는 삭제 버튼을 클릭합니다.

info 삭제 버튼 위치 안내
2026년 봄맞이 꿀벌 체험 프로그램 안내 프로그램
edit 수정 delete 삭제

※ 각 글 오른쪽의 빨간 '삭제' 버튼을 클릭합니다.

photo_camera [스크린샷] 게시글 목록 — 삭제 버튼 위치 삭제 버튼에 빨간 동그라미 표시
C-3

삭제 확인하기

①

삭제 버튼을 클릭하면 "정말 삭제하시겠습니까?"라는 확인 창이 나타납니다.

②

확인 창에서 선택하세요:

error_outline

정말 삭제하시겠습니까?

이 글을 삭제하면 되돌릴 수 없습니다.

취소 삭제 확인
삭제 확인 — 글이 완전히 삭제됩니다
취소 — 삭제를 취소하고 돌아갑니다
③

삭제 확인을 클릭하면 "삭제되었습니다" 메시지가 나타나고, 글이 목록에서 사라집니다.

photo_camera [스크린샷] 삭제 확인 팝업 창 '삭제 확인' 버튼에 빨간 동그라미 표시
C-4

삭제 후 복구 가능 여부

error

삭제한 글은 복구할 수 없습니다.

비온팜 시스템에서는 한 번 삭제된 글을 되살리는 기능을 제공하지 않습니다. 따라서 삭제 전에 반드시 다시 한번 확인해 주세요.

tips_and_updates

꿀팁: 삭제 대신 "비공개" 활용하기

글을 완전히 지우고 싶지 않다면, 글의 공개 상태를 "비공개"로 변경해 보세요. 비공개로 설정하면 홈페이지에는 보이지 않지만, 관리자 화면에서는 계속 확인할 수 있습니다. 나중에 다시 "공개"로 바꾸면 홈페이지에 다시 표시됩니다.

visibility 공개 arrow_forward visibility_off 비공개 로 변경
info

글 삭제 절차 요약

게시글 관리 → 삭제할 글 찾기 → 삭제 버튼 클릭 → 확인 창에서 '삭제 확인' 클릭

tips_and_updates

유용한 팁 & FAQ

콘텐츠 관리와 관련하여 자주 묻는 질문과 유용한 팁을 모아두었습니다.

Q

글을 저장했는데 홈페이지에 안 보여요.

A

두 가지를 확인해 주세요:
① 홈페이지를 새로고침(키보드 F5 키)해 보세요.
② 글의 상태가 "공개"로 되어 있는지 확인해 주세요. "비공개"나 "임시저장" 상태인 글은 홈페이지에 표시되지 않습니다.

Q

글을 수정했는데 이전 내용으로 보여요.

A

브라우저가 이전 페이지를 기억하고 있을 수 있습니다. Ctrl + Shift + R 키를 동시에 눌러 "강력 새로고침"을 해 보세요. 그러면 최신 내용이 표시됩니다.

Q

사진 파일이 업로드가 안 돼요.

A

다음 사항을 확인해 주세요:
① 파일 크기가 5MB 이하인지 확인합니다.
② 파일 형식이 JPG, PNG, WebP 중 하나인지 확인합니다.
③ 인터넷 연결이 정상인지 확인합니다.
위 방법으로도 해결되지 않으면 관리자에게 문의해 주세요.

Q

삭제한 글을 되살릴 수 있나요?

A

안타깝지만, 한 번 삭제한 글은 되살릴 수 없습니다. 삭제하기 전에 정말 지워도 괜찮은지 꼭 확인해 주세요. 중요한 글은 삭제 대신 "비공개"로 상태를 바꿔 숨기는 것을 추천합니다.

Q

작성 중에 실수로 화면을 닫아버렸어요.

A

아직 저장하지 않은 글은 안타깝지만 복구할 수 없습니다. 중요한 내용을 작성할 때는 중간중간 임시 저장 버튼을 눌러 저장해두는 것이 좋습니다.

Q

사진 크기가 5MB를 넘어요. 어떻게 하나요?

A

사진 파일 크기를 줄여야 합니다. 두 가지 방법이 있습니다:
① 컴퓨터에서 사진을 오른쪽 클릭 → 편집하여 크기를 줄입니다.
② 무료 이미지 압축 사이트(예: iloveimg.com)를 이용하여 사진 파일 크기를 줄인 뒤 다시 올립니다.
