사이트 접속 방법

회원가입 정보 입력

회원가입 버튼을 클릭하면 정보를 입력하는 화면이 나타납니다.

3-1

입력해야 할 항목 안내

회원가입 화면에는 다음 4가지 항목을 입력해야 합니다. 별표(*)가 표시된 항목은 반드시 입력해야 합니다.

person 이름 * (필수)

본인의 이름을 입력합니다.
예시: 홍길동

email 이메일 주소 * (필수)

실제 사용하는 이메일 주소를 입력합니다. 로그인할 때 이 이메일을 사용합니다.
예시: hong@gmail.com

lock 비밀번호 * (필수)

6자 이상의 비밀번호를 만들어 입력합니다. 자세한 규칙은 4. 비밀번호 규칙 안내를 참고하세요.
예시: myBee2024!

phone 전화번호 (선택)

연락 가능한 전화번호를 입력합니다. 입력하지 않아도 가입이 가능합니다.
예시: 010-1234-5678

5

가입 완료 확인

가입하기 버튼을 클릭한 후, 가입이 제대로 되었는지 확인하는 방법입니다.

5-1

가입 성공 시 화면

모든 정보를 올바르게 입력하고 가입하기를 누르면, 아래와 같은 성공 메시지가 나타납니다.

beeonfarm.co.kr/register
check_circle

회원가입이 완료되었습니다!

비온팜에 오신 것을 환영합니다.

로그인하러 가기
check 이 화면이 나오면 가입에 성공한 것입니다. "로그인하러 가기"를 눌러 로그인하세요.
5-2

가입이 안 될 때 (오류 메시지)

가입 중에 문제가 있으면 아래와 같은 오류 메시지가 나타납니다. 메시지를 읽고 해당 항목을 수정해 주세요.

error

"이메일과 비밀번호는 필수입니다."

해결: 이메일 주소와 비밀번호 칸이 비어있지 않은지 확인하세요.

error

"이름은 필수입니다."

해결: 이름 칸에 본인의 이름을 입력하세요.

error

"비밀번호는 6자 이상이어야 합니다."

해결: 비밀번호를 6자 이상으로 다시 입력하세요.

error

"유효한 이메일 형식이 아닙니다."

해결: 이메일 주소에 @ 기호가 포함되어 있는지 확인하세요. (예: hong@gmail.com)

error

"이미 등록된 이메일입니다."

해결: 이미 같은 이메일로 가입한 적이 있습니다. 다른 이메일을 사용하거나, 로그인을 시도해 보세요.

로그인 실패 시 대처법

로그인이 되지 않을 때, 원인별로 해결 방법을 안내합니다.

7-1

비밀번호가 틀렸을 때

오류 메시지: "이메일 또는 비밀번호가 올바르지 않습니다."

error_outline

이메일 또는 비밀번호가 올바르지 않습니다.

입력한 정보를 다시 확인해 주세요.

1

Caps Lock 확인하기

키보드의 Caps Lock 키가 켜져 있으면 대문자로 입력됩니다. Caps Lock 불빛이 꺼져 있는지 확인하세요.

2

한/영 전환 확인하기

비밀번호를 영어로 설정했는데 한글로 입력하고 있을 수 있습니다. 화면 오른쪽 아래에서 입력 언어가 "영어(EN)"인지 확인하세요.

3

비밀번호 다시 입력하기

비밀번호 칸의 내용을 모두 지우고, 천천히 다시 입력해 보세요. 눈 아이콘 visibility을 눌러 입력한 비밀번호를 확인할 수도 있습니다.

7-2

info

7-4

화면이 안 뜨거나 오류가 계속 날 때

1

2

새로고침 해보기

키보드에서 F5 키를 누르거나, 브라우저의 새로고침 버튼 refresh을 클릭하세요.

3

잠시 후 다시 시도

서버에 일시적인 문제가 있을 수 있습니다. 5~10분 후에 다시 시도해 보세요.

Q

비밀번호를 잊어버렸을 때 어떻게 하나요?

A

  2. 2 가입할 때 사용한 이메일 주소를 입력합니다.
  3. 3 이메일로 전송된 비밀번호 재설정 링크를 클릭합니다.
  4. 4 새로운 비밀번호를 입력하고 "변경" 버튼을 누릅니다.

Q

이미지가 업로드되지 않을 때 어떻게 하나요?

A

이미지 업로드가 안 될 때는 아래 사항을 확인해 주세요.

image
파일 형식 확인

지원되는 형식: JPG PNG GIF WEBP

BMP, TIFF 등 다른 형식은 업로드되지 않을 수 있습니다.

scale
파일 크기 확인

최대 업로드 용량은 5MB입니다. 사진이 너무 크면 이미지 압축 사이트(예: tinypng.com)를 이용해 줄여 주세요.

wifi
인터넷 연결 확인

Wi-Fi나 데이터 연결이 불안정하면 업로드가 중단될 수 있습니다. 네트워크 상태를 확인 후 다시 시도해 주세요.

beeonfarm.co.kr/admin/upload
cloud_upload

이미지를 여기에 끌어다 놓거나 클릭하여 선택

error

파일 크기가 5MB를 초과합니다. 더 작은 파일을 선택해 주세요.

Q

작성한 글이 화면에 보이지 않을 때 어떻게 하나요?

A

글을 작성했는데 화면에 보이지 않는 경우, 다음을 확인해 주세요.

1
페이지 새로고침

브라우저에서 F5 키를 누르거나, 주소창 옆의 refresh 버튼을 클릭하세요. 캐시된 이전 페이지가 보이고 있을 수 있습니다.

2
"저장" 또는 "게시" 버튼 확인

글 작성 후 반드시 "저장" 또는 "게시" 버튼을 눌러야 합니다. 버튼을 누르지 않으면 글이 반영되지 않습니다.

3
"공개" 상태 확인

글이 "비공개" 또는 "임시저장" 상태로 되어 있으면 다른 사람에게는 보이지 않습니다. 관리자 페이지에서 공개 상태를 확인하세요.

beeonfarm.co.kr/admin/posts
게시물 관리
꿀벌 체험 프로그램 안내 공개
봄 시즌 할인 이벤트 비공개 arrow_back

photo_camera 게시물 상태 확인 — "비공개" 상태의 글은 방문자에게 보이지 않습니다

Q

브라우저가 이상하게 보일 때 어떻게 하나요?

A

화면이 깨지거나 버튼이 작동하지 않을 때는 아래 방법을 시도해 보세요.

cached
캐시 및 쿠키 삭제

브라우저에 저장된 오래된 데이터가 문제를 일으킬 수 있습니다.
Ctrl + Shift + Delete (Windows) 또는 Cmd + Shift + Delete (Mac)를 눌러 캐시를 삭제해 주세요.

update
브라우저 최신 버전으로 업데이트

오래된 브라우저에서는 사이트가 제대로 표시되지 않을 수 있습니다. 항상 최신 버전을 유지해 주세요.
