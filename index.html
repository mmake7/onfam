파트 A: 사용자 매뉴얼

회원가입부터 로그인까지 — 7개 섹션

A-1

사이트 접속 방법

비온팜 홈페이지에 접속하는 방법을 알려드립니다.
인터넷 브라우저(크롬, 엣지 등)만 있으면 누구나 접속할 수 있습니다.

1-1

인터넷 브라우저 열기

컴퓨터 바탕화면이나 작업 표시줄에서 인터넷 브라우저 아이콘을 찾아 두 번 클릭합니다.

info 추천 브라우저
크롬 (Chrome) 엣지 (Edge) 파이어폭스 (Firefox)
1-2

홈페이지 주소 입력하기

브라우저 맨 위 주소 입력란(긴 흰색 막대)을 클릭한 뒤, 아래 주소를 입력하고 Enter 키를 누릅니다.

https://www.beeonfarm.co.kr

※ 주소를 정확히 입력해야 합니다. 오타가 나면 페이지가 열리지 않습니다.

photo_camera [스크린샷 자리] 브라우저 주소창 예시 화면
A-2

회원가입 버튼 찾기

홈페이지에 접속했다면, 이제 회원가입 화면으로 이동합니다.
화면에서 회원가입 버튼이 어디에 있는지 찾아볼게요.

2-1

상단 메뉴에서 회원가입 버튼 찾기

홈페이지 맨 위(상단 메뉴바)를 보세요. 오른쪽에 회원가입 또는 참여 신청 버튼이 있습니다. 이 버튼을 클릭하세요.

2-2

휴대폰(모바일)에서 접속한 경우

휴대폰으로 접속하면 메뉴가 숨겨져 있을 수 있습니다. 오른쪽 위의 menu (가로줄 세 개) 아이콘을 누르면 메뉴가 펼쳐집니다.

photo_camera [스크린샷 자리] 상단 메뉴 바 예시 화면
A-3

회원가입 정보 입력

회원가입 버튼을 클릭하면 정보를 입력하는 화면이 나타납니다.
아래 안내에 따라 각 항목을 하나씩 입력해 주세요.

3-1

입력해야 할 항목 안내

회원가입 화면에는 다음 4가지 항목을 입력해야 합니다. 별표(*)가 표시된 항목은 반드시 입력해야 합니다.

person이름 *(필수)

본인의 이름을 입력합니다.
예시: 홍길동

email이메일 주소 *(필수)

실제 사용하는 이메일 주소를 입력합니다. 로그인할 때 이 이메일을 사용합니다.
예시: hong@gmail.com

lock비밀번호 *(필수)

6자 이상의 비밀번호를 만들어 입력합니다. 자세한 규칙은 A-4. 비밀번호 규칙 안내를 참고하세요.
예시: myBee2024!

phone전화번호(선택)

연락 가능한 전화번호를 입력합니다. 입력하지 않아도 가입이 가능합니다.
예시: 010-1234-5678

photo_camera [스크린샷 자리] 회원가입 양식 화면
A-4

비밀번호 규칙 안내

안전한 비밀번호를 만드는 규칙을 알려드립니다.
규칙을 지키지 않으면 가입이 되지 않으니, 아래 내용을 꼭 확인하세요.

4-1

비밀번호 필수 규칙

check_circle

6자 이상이어야 합니다

너무 짧은 비밀번호는 보안에 취약하기 때문입니다.

더 안전한 비밀번호를 만드는 팁

tips_and_updates

영어 대/소문자 + 숫자를 섞으면 더 안전합니다

예: MyBee2024

tips_and_updates

특수문자(!@#$)를 넣으면 더욱 안전합니다

예: MyBee2024!

4-2

비밀번호 예시 (좋은 예 / 나쁜 예)

check_circle좋은 비밀번호
BeeHoney2024!
Farm#Love99
MyBee@Home7
cancel나쁜 비밀번호
123455자 (너무 짧음)
abcde5자 (너무 짧음)
abc3자 (너무 짧음)
warning

비밀번호는 꼭 기억해 주세요!

비밀번호를 잊어버리면 로그인을 할 수 없습니다. 비밀번호를 메모장이나 수첩에 적어두시는 것을 추천합니다. 단, 다른 사람이 볼 수 없는 안전한 곳에 보관하세요.

A-5

가입 완료 확인

가입하기 버튼을 클릭한 후, 가입이 제대로 되었는지 확인하는 방법입니다.

5-1

가입 성공 시 화면

모든 정보를 올바르게 입력하고 가입하기를 누르면, “회원가입이 완료되었습니다!”라는 성공 메시지가 나타납니다.

5-2

가입이 안 될 때 (오류 메시지)

가입 중에 문제가 있으면 아래와 같은 오류 메시지가 나타납니다. 메시지를 읽고 해당 항목을 수정해 주세요.

error

“이메일과 비밀번호는 필수입니다.”

해결: 이메일 주소와 비밀번호 칸이 비어있지 않은지 확인하세요.

error

“이름은 필수입니다.”

해결: 이름 칸에 본인의 이름을 입력하세요.

error

“비밀번호는 6자 이상이어야 합니다.”

해결: 비밀번호를 6자 이상으로 다시 입력하세요.

error

“유효한 이메일 형식이 아닙니다.”

해결: 이메일 주소에 @ 기호가 포함되어 있는지 확인하세요. (예: hong@gmail.com)

error

“이미 등록된 이메일입니다.”

해결: 이미 같은 이메일로 가입한 적이 있습니다. 다른 이메일을 사용하거나, 로그인을 시도해 보세요.

A-6

로그인하기

회원가입이 완료되었다면, 이제 로그인을 해 볼 차례입니다.
가입할 때 사용한 이메일과 비밀번호를 입력하면 됩니다.

6-1

로그인 화면으로 이동하기

홈페이지 상단 메뉴에서 로그인 글자를 클릭합니다.

6-2

이메일과 비밀번호 입력하기

로그인 화면이 나타나면, 가입할 때 사용한 이메일 주소와 비밀번호를 입력합니다.

6-3

로그인 성공 확인하기

로그인에 성공하면 다음과 같은 변화가 나타납니다:

1

상단 메뉴의 “로그인” 글자가 내 이름으로 바뀝니다.

2

메인 화면(홈페이지) 또는 대시보드로 자동 이동합니다.

3

로그아웃 버튼이 새로 나타납니다.

photo_camera [스크린샷 자리] 로그인 화면 예시
A-7

로그인 실패 시 대처법

로그인이 되지 않을 때, 원인별로 해결 방법을 안내합니다.
당황하지 말고 아래 내용을 하나씩 확인해 보세요.

7-1

비밀번호가 틀렸을 때

오류 메시지: “이메일 또는 비밀번호가 올바르지 않습니다.”

1

Caps Lock 확인하기

키보드의 Caps Lock 키가 켜져 있으면 대문자로 입력됩니다.

2

한/영 전환 확인하기

비밀번호를 영어로 설정했는데 한글로 입력하고 있을 수 있습니다.

3

비밀번호 다시 입력하기

비밀번호 칸의 내용을 모두 지우고, 천천히 다시 입력해 보세요.

7-2

이메일을 잘못 입력했을 때

1

이메일 주소에 오타가 없는지 확인

흔한 실수: hong@gmai.com (X) → hong@gmail.com (O)

2

다른 이메일로 가입한 건 아닌지 확인

가입할 때 사용한 이메일과 로그인할 때 입력한 이메일이 같은지 확인해 보세요.

7-3

아직 회원가입을 하지 않은 경우

info

로그인하려면 먼저 회원가입이 필요합니다. 아직 가입하지 않았다면 A-2. 회원가입 버튼 찾기부터 다시 시작해 주세요.

7-4

화면이 안 뜨거나 오류가 계속 날 때

1

인터넷 연결 확인

Wi-Fi 또는 인터넷이 정상적으로 연결되어 있는지 확인하세요.

2

새로고침 해보기

키보드에서 F5 키를 누르거나, 브라우저의 새로고침 버튼을 클릭하세요.

3

잠시 후 다시 시도

서버에 일시적인 문제가 있을 수 있습니다. 5~10분 후에 다시 시도해 보세요.

4

관리자에게 문의

위 방법으로도 해결이 안 되면 관리자에게 연락해 주세요.

admin_panel_settings

파트 B: 관리자 매뉴얼

대시보드부터 콘텐츠 관리까지 — 6개 섹션

B-1

관리자 화면 접속

비온팜 홈페이지와 관리자 화면에 접속하는 방법을 알려드립니다.
인터넷 브라우저(크롬, 엣지 등)만 있으면 누구나 접속할 수 있습니다.

B1-1

홈페이지 주소 입력하기

브라우저 주소 입력란에 아래 주소를 입력하고 Enter 키를 누릅니다.

https://www.beeonfarm.co.kr
B1-2

관리자 화면으로 이동하기

홈페이지가 열리면, 주소 뒤에 /admin을 추가로 입력하여 관리자 전용 화면으로 이동합니다.

https://www.beeonfarm.co.kr/admin
warning

주의: 관리자 화면은 권한이 있는 사람만 접속할 수 있습니다. 로그인이 필요합니다.

photo_camera [스크린샷 자리] 브라우저 주소창 예시 화면
B-2

관리자 로그인

관리자 화면을 사용하려면 먼저 로그인해야 합니다.
이미 로그인 방법을 아시면 B-3. 대시보드 사용법으로 넘어가세요.

B2-1

로그인하기

⊕

로그인 화면에서 가입할 때 사용한 이메일 주소와 비밀번호를 입력합니다.

⊕

로그인 버튼을 클릭합니다.

⊕

관리자 대시보드 화면이 나타나면 로그인 성공입니다.

B2-2

비밀번호를 잊어버렸을 때

⊕

로그인 화면에서 비밀번호 찾기 글자를 클릭합니다.

⊕

가입할 때 사용한 이메일 주소를 입력하고 전송을 누릅니다.

⊕

이메일을 확인하여 안내에 따라 새 비밀번호를 설정합니다.

photo_camera [스크린샷 자리] 로그인 화면 예시
B-3

관리자 대시보드 사용법

로그인하면 가장 먼저 보이는 화면이 대시보드입니다.
대시보드에서는 홈페이지의 전체 현황을 한눈에 볼 수 있습니다.

B3-1

대시보드 화면 구성 알아보기

대시보드는 크게 네 부분으로 나뉩니다.

menu왼쪽 메뉴바

여러 관리 기능으로 이동할 수 있는 메뉴 목록입니다.

bar_chart통계 요약

전체 게시글 수, 방문자 수 등을 숫자로 보여줍니다.

list_alt최근 게시글

가장 최근에 작성된 글 목록이 보입니다.

person내 정보

오른쪽 위에 로그인된 이름과 로그아웃 버튼이 있습니다.

B3-2

왼쪽 메뉴 사용하기

왼쪽 메뉴에서 원하는 항목을 클릭하면 해당 관리 화면으로 이동합니다.

dashboard
대시보드— 전체 현황 화면 (첫 화면)
article
게시글 관리— 글 목록 보기, 새 글 쓰기
photo_library
이미지 관리— 사진 올리기, 삭제하기
settings
설정— 사이트 기본 정보 변경
photo_camera [스크린샷 자리] 관리자 대시보드 전체 화면
B-4

게시글 작성 / 수정 / 삭제

홈페이지에 보여지는 글(게시글)을 새로 쓰거나, 이미 올린 글의 내용을 고치거나, 필요 없는 글을 지우는 방법을 알려드립니다.

add_circle 새 글 쓰기 edit 글 수정하기 delete 글 삭제하기
B4-1

새 글 쓰기

홈페이지에 새로운 글을 올리는 방법

⊕

왼쪽 메뉴에서 게시글 관리를 클릭합니다.

⊕

화면 오른쪽 위의 + 새 글 쓰기 버튼을 클릭합니다.

⊕

글 작성 화면이 나타납니다. 아래 항목을 채워주세요:

제목— 글의 제목을 입력합니다
카테고리— 글의 종류를 선택합니다
내용— 본문 내용을 작성합니다
대표 이미지— 글과 함께 보일 사진을 선택합니다 (선택사항)
⊕

모든 내용을 입력한 뒤, 저장하기 버튼을 클릭합니다.

B4-2

글 수정하기

이미 올린 글의 내용을 고치는 방법

⊕

B-6

자주 묻는 질문 (FAQ)

관리자들이 자주 질문하는 내용을 모아두었습니다.
아래에서 비슷한 상황을 찾아 해결 방법을 확인해 보세요.

Q

비밀번호가 기억나지 않아요.

A

로그인 화면에서 “비밀번호 찾기”를 클릭하고, 가입할 때 사용한 이메일 주소를 입력해 주세요. 이메일로 비밀번호 재설정 안내가 발송됩니다. 자세한 방법은 B-2. 관리자 로그인을 참고하세요.

Q

글을 저장했는데 홈페이지에 안 보여요.

A

먼저 홈페이지를 새로고침(키보드 F5 키 또는 Ctrl+R)해 보세요. 그래도 안 보인다면, 글의 상태가 “공개”로 되어 있는지 확인해 주세요. “비공개”로 설정된 글은 홈페이지에 표시되지 않습니다.

Q

사진이 업로드가 안 돼요.

A

다음 사항을 확인해 주세요:
• 파일 크기가 5MB 이하인지 확인합니다.
• 파일 형식이 JPG, PNG, WebP 중 하나인지 확인합니다.
• 인터넷 연결이 정상인지 확인합니다.
위 방법으로도 해결되지 않으면 관리자에게 문의해 주세요.

Q

삭제한 글을 되살릴 수 있나요?

A

안타깝지만, 한 번 삭제한 글은 되살릴 수 없습니다. 삭제하기 전에 정말 지워도 괜찮은지 꼭 확인해 주세요. 중요한 글은 삭제 대신 “비공개”로 상태를 바꿔 숨기는 것을 추천합니다.

Q

관리자 화면이 갑자기 안 열려요.

A

• 인터넷 연결이 되어 있는지 확인합니다.
• 브라우저의 주소가 정확한지 다시 한번 확인합니다.
• 브라우저를 완전히 껐다가 다시 열어봅니다.
• 그래도 안 된다면 다른 브라우저(크롬, 엣지)로 시도해 보세요.

Q

글을 수정했는데 이전 내용으로 보여요.

A

브라우저가 이전 페이지를 기억하고 있을 수 있습니다. Ctrl + Shift + R 키를 동시에 눌러 “강력 새로고침”을 하면 최신 내용이 표시됩니다.
